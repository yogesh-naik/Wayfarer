## Psql Database Setup
* install psql
* run command `psql` in terminal
* `CREATE DATABASE nitelife;`


## How to Open NitelLife App in Development:

 * Clone Repo
 * `pipenv shell`
 * `pipenv sync`
 * `python3 manage.py makemigrations`
 * `python3 manage.py migrate`
 * `python3 manage.py runserver`
